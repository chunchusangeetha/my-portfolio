## Sangeetha's Portfolio

A modern and responsive Next.js portfolio website showcasing my projects, skills, and experience as a Frontend Developer.

## Getting Started

1.Git clone https://github.com/chunchusangeetha/my-portfolio.git

2.npm install 

3.run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000]
